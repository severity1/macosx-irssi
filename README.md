macosx-irssi
============

solarized irssi configuration for macosx. Tested on Mavericks.

##Installation
- Install irssi

```bash
brew install irssi
```

- Install solarized-universal-theme from this repo.

```url
https://github.com/huyz/irssi-colors-solarized
```

- Replace .irssi/config with the one here.
